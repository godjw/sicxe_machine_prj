프로젝트 1에서 구현한 shell에 assemble 기능을 추가했다.
.asm파일을 입력받아 lst 파일과 obj 파일을 생성한다. windows에서 asm 파일을 생성하여
^M과 같은 문자가 저장되어 형식이 달라지는 경우는 오류 처리를 해주지 못하였다.

-컴파일 방법
소스코드와 Makefile이 있는 directory에서 make를 입력하여 컴파일 할 수 있다.
이후 생성되는 20161643.out 파일을 ./20161643.out을 통해서 실행한다. 
주어진 명령어를 입력하여 실행 결과를 화면에서 확인 할 수 있다. 

-삭제 방법
make clean으로 2016143.o, 20161643.out, .lst, .obj, intermediate 파일을 삭제할 수 있다
